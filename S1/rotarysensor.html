ROTARY SENSOR

HOW TO USE SENSORS FOR DATA COLLECTION

The sensor (s) are already connected.

DO NOT DISTURB THESE CONNECTIONS.

  1. Open the Logger Pro - 3 software.

  2. On the left-hand corner observe if the sensor reading (s) are displayed.

  3. Slowly move the mass (es) and ascertain that the sensor reading (s) are changing.

  4. Zero the sensor (s) position (s). For this, bring the object (s) to rest and then zero it.

  5. Press the 'watch' icon. Here, the settings of data collection can be changed.

a. Put a value of 7-10 for 'no of points/second ' to be collected.

b. Put the time of collection about 10 -20 s.

  1. Initiate the motion of the system, and wait for a couple of oscillations to occur. Then press the data collection (play) button. This will start collecting the data.

  2. Use 'Cntrl+j' to automatically set the axes of the graph.
